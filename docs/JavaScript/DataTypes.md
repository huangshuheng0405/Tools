# 数据类型

## 基本数据类型

- 数字（Number） : 包括整数和浮点数
- 字符串（String） : 文本数据，使用单引号或双引号括起来
- 布尔值（Boolean） : 只有两个值 true 和 false
- 空值（Null） : 表示一个空值或没有值的对象
- 未定义（Undefined） : 表示一个未被赋值的变量的值
- 符号（Symbol） : 唯一的标识符，用于对象属性的访问

## 引用数据类型

- 对象（Object） : 复杂数据类型，用于存储多个值和属性
- 数组（Array） : 特殊的对象，用于存储有序的多个值
- 函数（Function） : 可执行的代码块，用于封装可重复使用的逻辑

ECMAScript 2020 (ES11) 规范中正式添加了一种新的数据类型：

- 大整数（BigInt） : 用于表示大于 2^53 - 1 的整数

## tips

基础类型存储在**栈内存**中，变量记录原始值，引用类型存储在**堆内存**中，变量记录引用地址。
