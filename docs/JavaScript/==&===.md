# == 和 === 的区别

- `==` 比较值是否相等，**会进行类型转换**
- `===` 比较值是否相等，**不会进行类型转换**

```javascript
123 == '123' // true 在比较之前 字符串'123'会被转换为数字123
true == 1 // true 布尔值true会被转换为数字1
0 == false
"0" == false // true 字符串"0"会被转换为数字0
[] == false
```

```javascript
123 === '123' // false 不会进行类型转换
true === 1 // false 不会进行类型转换
null === undefined // false 不会进行类型转换
```

```javascript
// 加什么使下面代码成立

if (a == 1 && a == 2 && a == 3) {
  console.log('a == 1 && a == 2 && a == 3')
}
```

答案

```javascript
const a = {
  i: 1,

  valueOf: function () {
    return this.i++
  }
}

if (a == 1 && a == 2 && a == 3) {
  console.log('a == 1 && a == 2 && a == 3')
}
```
